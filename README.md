# EjdEdart (v1.0)

En mobil-first webbapp (PWA) för dart: 301/501, 1–4 spelare, set/legs, coin toss start, snabb inmatning via cirkel-grid, checkout-förslag, bust-logik och undo.

## Starta lokalt
Eftersom Service Worker kräver en server, kör via en enkel lokal server:

- Python: `python -m http.server 8000`
- Öppna: `http://localhost:8000`

På mobil: öppna URL och välj **“Lägg till på hemskärmen”** (iOS/Android) för fullscreen/standalone-läge.
